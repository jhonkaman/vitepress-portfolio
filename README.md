# VitePress Portfolio Project

## Commands

### Start dev server

```
npm run docs:dev
```

### Build

```
npm run docs:build
```

### Preview build

```
npm run docs:preview
```

## Create

Here are the instructions for recreating this project.

### Create folder

```
mkdir vitepress-portfolio
```

```
cd vitepress-portfolio
```

### Add package

```
npm init -y
```

```
npm add -D vitepress@next
```

### Setup

```
npx vitepress init
```
